Ingeniería de Software III — Laboratorios

Repositorio público de guías y laboratorios (Spring Boot / Arquitectura / Buenas prácticas).

Cómo usar este sitio

  1. Selecciona una pestaña (Lab 01, Lab 02, …).
  2. Abre el laboratorio completo desde el enlace del panel.
  3. Para agregar nuevos laboratorios:
    • Crea un archivo HTML en /labs/lab-XX.html
    • Actualiza esta portada para incluir la nueva pestaña + enlace.

Sugerencia: mantén un nombre consistente: lab-01.html, lab-02.html, etc.

Estructura recomendada

/index.html
/labs
  lab-01.html
  lab-02.html
  lab-03.html
/assets
  (opcional: imágenes, css)

GitHub Pages publica automáticamente el contenido del repositorio. El archivo index.html es la portada.
